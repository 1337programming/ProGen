# ProGen
You'd think it's for protractor... but it's actually a ad for a Gillette shaving cream commercial... written in Angular 2 

# Install
npm install

# Build application
npm run build

# Test application

# run dev server localhost:3333
npm start

# build and open electron preview
npm run electron

# Create electron package
npm run package
